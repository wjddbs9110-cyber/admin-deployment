# 📘 광고 솔루션 어드민

------------------------------------------------------------------------

## 🧭 프로젝트 개요

**프로젝트명:** 광고 솔루션 어드민 (Marketer Admin Console)\
**목표:** 광고 캠페인 운영, 성과 분석, 예산 통제, 소재 관리 등 마케팅
업무를 통합 관리할 수 있는 플랫폼 구축\
**주요 사용자:** 마케팅 매니저, 퍼포먼스 마케터, 데이터 애널리스트

------------------------------------------------------------------------

## ⚙️ 주요 메뉴 구조 (3Depth)

**대시보드**\
- 실시간 리포트\
- 캠페인 요약\
- 예산 현황

- 전체 캠페인\

- 캠페인 생성\
- 캠페인 템플릿 관리

**광고 소재 관리**\
- 소재 리스트\
- 소재 업로드\
- A/B 테스트 관리

**타겟팅 & 오디언스**\
- 오디언스 세그먼트\
- 리타겟팅 설정\
- 맞춤 타겟 생성

**리포트 & 분석**\
- 성과 분석\
- 전환 추적\
- 다운로드 리포트

**결제 & 예산**\
- 충전 내역\
- 예산 한도 설정\
- 청구서 관리

**시스템 설정**\
- 팀원 관리\
- 권한 설정\
- 계정 연동 (Google, Meta 등)\
- 로그 내역

**사이드바 하단 구성:**\
- 🔔 알림 버튼 (배지 표시)\
- 👤 유저 메뉴 (프로필, 비밀번호 변경, 로그아웃)

## 🎯 전체 캠페인 관리 화면 구성 요소

  영역명                 구성 요소                                       설명
  ---------------------- ----------------------------------------------- ---------------------------------
  **필터 영역**          검색어, 상태, 채널, 날짜 범위, 카테고리         캠페인 검색 및 필터링
  (CardFilter)

  **액션 버튼 영역**     캠페인 생성, 선택 항목 수정, 선택 항목 삭제,    상단 우측 버튼 그룹
  (headerRight)          엑셀 다운로드, 새로고침

  **캠페인 목록**        체크박스, 캠페인명, 상태(Tag), 채널,            전체 캠페인 목록 표시
  (CardTable)            예산, 노출수, 클릭수, CTR, ROAS,                정렬, 페이지네이션 지원
                         생성일, 액션(수정/삭제/복사)

  **페이지네이션**       페이지 번호, 페이지당 행 수 선택                 우측 정렬

### 필터 항목 상세

  필터명       입력 타입         옵션                                            설명
  ------------ ----------------- ----------------------------------------------- -------------------------
  검색어       Input             placeholder: "캠페인명 검색"                    캠페인명으로 검색

### 테이블 컬럼 상세

  컬럼명       타입       정렬    필터    설명
  ------------ ---------- ------- ------- -----------------------------------------
  체크박스     Checkbox   -       -       일괄 작업용 선택
  캠페인명     Text       ○       -       클릭 시 상세 페이지 이동
  상태         Tag        ○       ○       활성(green), 일시정지(orange), 종료(red)
  채널         Tag        ○       ○       Google, Meta, Kakao 등
  예산         Number     ○       -       일 예산 / 총 예산 표시
  노출수       Number     ○       -       천단위 구분 표시
  클릭수       Number     ○       -       천단위 구분 표시
  CTR          Percentage ○       -       클릭률 (소수점 2자리)
  ROAS         Percentage ○       -       광고 수익률 (소수점 2자리)
  생성일       Date       ○       -       YYYY-MM-DD 형식
  액션         Button     -       -       수정/삭제/복사 버튼

### 주요 기능

1. **검색 및 필터링**
   - 실시간 검색 (debounce 300ms)
   - 다중 필터 조합 가능
   - 필터 초기화 버튼

2. **정렬**
   - 모든 숫자/날짜 컬럼 정렬 가능
   - 오름차순/내림차순 토글

3. **일괄 작업**
   - 체크박스로 다중 선택
   - 선택 항목 일괄 수정 (상태 변경, 예산 조정)
   - 선택 항목 일괄 삭제 (확인 모달)

4. **상세 액션**
   - 캠페인명 클릭 → 상세 페이지
   - 수정 버튼 → 수정 모달/페이지
   - 삭제 버튼 → 확인 모달 → 삭제
   - 복사 버튼 → 이름 변경 모달 → 복사 생성

5. **데이터 내보내기**
   - 엑셀 다운로드 (현재 필터 적용된 데이터)
   - CSV 형식 지원

## 🎨 주요 화면 구조 (와이어프레임 기반)

**01. 로그인 화면**\
- 이메일, 비밀번호 입력, 로그인 버튼, 비밀번호 찾기 링크

**02. 메인 대시보드**\
- KPI 카드 4개, 트렌드 차트, 채널 비교, 예산 게이지, Top 캠페인

**03. 캠페인 리스트**\
- 필터, 테이블, 캠페인 생성 버튼, 상세 진입

**04. 캠페인 상세**\
- 성과/소재/오디언스/로그 탭 구조

**05. 소재 관리**\
- 썸네일형 리스트, 업로드, 태그 필터, 상태 토글

**06. 리포트 & 분석**\
- 다중 차트, 필터, 다운로드 버튼

**07. 알림 패널**\
- 슬라이드 인 구조, 실시간 알림 탭

**08. 유저 메뉴 팝오버**\
- 프로필, 알림 설정, 로그아웃 메뉴

## 🧩 컴포넌트 정의서
